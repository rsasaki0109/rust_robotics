# MovingAI Benchmark Subset

This directory vendors a small subset of the Moving AI Lab 2D grid benchmark set
for local planning benchmarks in `rust_robotics_planning`.

Included files:

- `dao/arena2.map`
- `dao/arena2.map.scen`
- `maze/maze512-1-0.map`
- `maze/maze512-1-0.map.scen`
- `room/8room_000.map`
- `room/8room_000.map.scen`
- `random/random512-10-0.map`
- `random/random512-10-0.map.scen`
- `street/Berlin_0_512.map`
- `street/Berlin_0_512.map.scen`

Source:

- `https://www.movingai.com/benchmarks/grids.html`
- `https://www.movingai.com/benchmarks/dao/dao-map.zip`
- `https://www.movingai.com/benchmarks/dao/dao-scen.zip`
- `https://www.movingai.com/benchmarks/maze/maze-map.zip`
- `https://www.movingai.com/benchmarks/maze/maze-scen.zip`
- `https://www.movingai.com/benchmarks/room/room-map.zip`
- `https://www.movingai.com/benchmarks/room/room-scen.zip`
- `https://www.movingai.com/benchmarks/random/random-map.zip`
- `https://www.movingai.com/benchmarks/random/random-scen.zip`
- `https://www.movingai.com/benchmarks/street/street-map.zip`
- `https://www.movingai.com/benchmarks/street/street-scen.zip`
