# rust_robotics_planning

Path planning algorithms for the RustRobotics workspace, including grid planners,
sampling-based planners, any-angle planners, trajectory generators, and coverage
planning utilities.

```toml
[dependencies]
rust_robotics_planning = "0.1"
```

```rust
use rust_robotics_planning::{AStarConfig, AStarPlanner, DWAPlanner};
```
