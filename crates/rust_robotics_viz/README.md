# rust_robotics_viz

Visualization helpers for RustRobotics examples and demos.

```toml
[dependencies]
rust_robotics_viz = "0.1"
```

This crate is intentionally small and is mostly used by the workspace examples
that render planner, controller, localization, and SLAM outputs.

Project site: https://rsasaki0109.github.io/rust_robotics/

Repository: https://github.com/rsasaki0109/rust_robotics
