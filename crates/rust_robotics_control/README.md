# rust_robotics_control

Control and path tracking algorithms for the RustRobotics workspace, including
Pure Pursuit, Stanley control, LQR, MPC, PID, manipulator control, behavior trees,
and mission state machines.

```toml
[dependencies]
rust_robotics_control = "0.1"
```

```rust
use rust_robotics_control::{PurePursuitController, StanleyController};
```
