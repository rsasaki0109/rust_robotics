# rust_robotics_mapping

Mapping algorithms for the RustRobotics workspace, including occupancy grids,
NDT maps, ray casting grid maps, Gaussian grid maps, point cloud sampling, and
geometric feature extraction.

```toml
[dependencies]
rust_robotics_mapping = "0.1"
```
