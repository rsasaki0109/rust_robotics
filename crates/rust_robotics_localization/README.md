# rust_robotics_localization

Localization algorithms for the RustRobotics workspace, including EKF, UKF,
particle filter, histogram filter, CKF, adaptive filters, and Monte Carlo
localization.

```toml
[dependencies]
rust_robotics_localization = "0.1"
```

```rust
use rust_robotics_localization::{EKFConfig, EKFLocalizer};
```
