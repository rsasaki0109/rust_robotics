# Security Policy

## Supported Versions

RustRobotics currently accepts security reports for the `main` branch.

## Reporting A Vulnerability

Please report security-sensitive issues privately to the repository owner instead
of opening a public issue. Include:

- the affected crate, example, script, or ROS2 node
- a minimal reproduction or command sequence
- expected impact and whether it requires untrusted input
- the Rust, OS, ROS2, or Gazebo versions involved

For ordinary correctness bugs, performance regressions, or documentation issues,
use the public issue templates.
